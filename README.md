# 🎓 Smart College Portal

A modern and responsive college website developed as a collaborative project.
The platform provides students and visitors with easy access to important notices, events, academic information, and campus resources.

---

## ✨ Features

* Welcome popup landing screen
* Important notice ribbon with downloadable PDFs
* Sticky navigation with dropdown menus
* Campus image slider
* Principal desk section
* Latest news & events
* College gallery (image / video / media)
* Footer with quick links and social media

---

## 🛠 Tech Stack

* HTML
* CSS
* JavaScript
* Git & GitHub collaboration

---

## 👨‍💻 Team

Developed by a group of students as part of a collaborative web development project.

---

## 🚀 Project Goal

To build a clean, structured, and user-friendly digital portal representing a modern college website.
